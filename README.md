# MM Thai Bus Guide

A cross-border bus travel guide app for Myanmar and Thailand!  
Features:  
- Provider state management
- i18n/localization (en, my, th) via ARB + Flutter localization
- Splash screen, custom app icon (add with flutter_launcher_icons)
- Bottom navigation (Route, Bus, Phrases, Destination, Settings)
- Dark mode support
- Persistent favorites (with SharedPreferences)
- Thai Text-to-Speech for routes/phrases
- Google Maps integration
- Ready for GitHub: MIT license, clean code
